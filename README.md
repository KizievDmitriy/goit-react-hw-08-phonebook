# React Phonebook App

Web application "Phonebook" with the ability to register users and work with
private collections of contacts.

## Core Technology Stack

- **React**
- **Redux**
- **SASS**
- **HTML**
- **JavaScript**
- **REST API** - (used backend: https://www.heroku.com/)

## Used Libraries

- **Axios**
- **Notiflix**
- **Material-UI**
- **Redux Toolkit**
- **RTK Query**
